# Job Roles In Robotics

### **Robotics Career Paths: Roles, Scope, and Industries** <a href="#undefined" id="undefined"></a>

Robotics is a multidisciplinary field offering a diverse range of career opportunities for individuals with varied skill sets. From designing and building robots to developing the AI that powers them and bringing these innovations to market, there's a role for many talents. This guide provides an overview of common career paths in robotics, their typical scope, and the industries that hire for each.

***

### **Robotics Career Roles**

| Role Title                                    | Scope                                                                                                  | Industries                                                              |
| --------------------------------------------- | ------------------------------------------------------------------------------------------------------ | ----------------------------------------------------------------------- |
| **Robotics Engineer**                         | Design, build, test, and maintain robotic systems and frameworks.                                      | Manufacturing, healthcare, aerospace, defense, research.                |
| **Robotics Technician**                       | Assembly, installation, maintenance, troubleshooting, and repair of robots.                            | Manufacturing, logistics, automation services.                          |
| **Mechanical Engineer (Robotics)**            | Structural design, mechanism design, thermal and mechanical analysis for robots.                       | Manufacturing, automotive, robotics R\&D, aerospace.                    |
| **Software Engineer (Robotics)**              | Develop software and algorithms that control robotic behavior, perception, navigation, AI integration. | Tech companies, automation, AI/ML startups, research, robotics firms.   |
| **Hardware Engineer (Robotics)**              | Design and test robot hardware: PCBs, processors, sensors, controllers, embedded systems.              | Embedded systems, electronics, robotics labs, IoT, tech (e.g., Amazon). |
| **Machine Learning / AI Engineer (Robotics)** | Develop algorithms for robot autonomy, perception (computer vision), decision-making, learning.        | Defense, autonomous vehicles, research, AI startups, tech.              |
| **Automation Engineer**                       | Design, program, simulate, and test automated machinery, often integrating robotic solutions.          | Automotive, packaging, electronics, food & beverage, pharma.            |
| **Design Engineer (Robotics)**                | Visual/functional design, CAD modeling, simulation, HRI aspects, ergonomics.                           | Industrial design, consumer robotics, R\&D.                             |
| **Sales Engineer (Robotics)**                 | Technical sales, client bridging, product demos, solution consulting for robotic systems.              | B2B robotics companies, automation vendors, tech distributors.          |
| **Aerospace Engineer (Robotics/UAV)**         | Design UAVs, drones, robotic systems for aircraft/spacecraft, satellite robotics.                      | Aerospace, defense, space exploration, drone manufacturers.             |
| **Computer Scientist (Robotics Research)**    | Develop core algorithms for perception, planning, control, data processing, real-time computing.       | Research institutions, AI firms, autonomous technology R\&D.            |

***

### **Specialized Modern Roles (2024-2026)**

Beyond the classical roles above, the post-foundation-model wave has created new specializations worth knowing:

| Role | What it does |
|---|---|
| **SLAM Engineer** | Builds and tunes SLAM stacks - filter / graph / visual / LiDAR / learned. See [SLAM & State Estimation](../slam-and-state-estimation/). |
| **Motion Planning Engineer** | Owns the Nav2 / MoveIt / custom planner stack. Behavior trees, controllers, costmaps. |
| **Perception Engineer** | Cameras, depth, LiDAR, foundation vision models (SAM, DINO, Depth Anything). |
| **Sensor Fusion Engineer** | EKF / UKF / factor-graph fusion across heterogeneous sensors. |
| **Robot Learning Engineer** | Imitation, RL, diffusion policies, VLAs. See [Robot Learning](../robot-learning/). |
| **Sim-to-Real Engineer** | Isaac Lab / MuJoCo MJX / Brax / Genesis; domain randomization; system identification. |
| **Foundation Model Engineer (Robotics)** | Trains and deploys VLAs (RT-2, OpenVLA, π0). |
| **Fleet / Orchestration Engineer** | Multi-robot coordination, scheduling, OTA updates, observability. |
| **ROS 2 Developer** | Builds and maintains the ROS 2 stack on a robot. See [ROS 2](../ros-2/). |
| **Robot SRE / DevOps** | CI/CD for robotics, rosbag replay infrastructure, on-robot observability. |
