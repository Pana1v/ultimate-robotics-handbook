# Fusion 360

### **Autodesk Fusion: A Comprehensive Guide for Design and Manufacturing in Robotics** <a href="#undefined" id="undefined"></a>

{% embed url="https://www.youtube.com/watch?pp=ygUKZnVzaW9uIDM2MA==&v=beebJ6fgVPo" %}

Autodesk Fusion (often still referred to by its former name, Fusion 360) is a cloud-based 3D modeling, CAD (Computer-Aided Design), CAM (Computer-Aided Manufacturing), CAE (Computer-Aided Engineering), and PCB (Printed Circuit Board) design software platform . Developed by Autodesk, it provides an integrated environment for product design and manufacturing, making it a popular choice for engineers, designers, machinists, and hobbyists, including those working on robotics projects . This guide will cover its key features, benefits, common applications in robotics, and resources for learning.

***

### **1. What is Autodesk Fusion?**

| Feature/Aspect            | Description                                                                                                                                                                                                                                                                                                               |
| ------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Core Functionality**    | A comprehensive, cloud-integrated platform for 3D design and manufacturing. It combines parametric, direct, surface, and mesh modeling, as well as simulation, rendering, CAM, and electronic design (ECAD/MCAD co-design) capabilities in a single software package [2](https://www.novatr.com/blog/what-is-fusion-360). |
| **Development & History** | Developed by Autodesk and launched in 2013 [2](https://www.novatr.com/blog/what-is-fusion-360). It has rapidly evolved, becoming a versatile tool for a wide range of design and manufacturing workflows.                                                                                                                 |
| **Cloud Integration**     | Being cloud-based allows for seamless collaboration among team members, version control, and access to projects from anywhere [2](https://www.novatr.com/blog/what-is-fusion-360). Designs are typically saved to the Autodesk cloud, though local saving is also possible 9.                                             |
| **Design Approach**       | Supports a "top-down" design approach, where users can create larger structures (assemblies) and then break them down into smaller, individually modifiable components [2](https://www.novatr.com/blog/what-is-fusion-360).                                                                                               |

### **4. Getting Started with Autodesk Fusion: User Interface Basics**

When you first open Fusion, you'll encounter several key areas 4, [8](https://cdn.instructables.com/ORIG/FRW/99T3/J12O325B/FRW99T3J12O325B.pdf), 12:

* **Application Bar:** Topmost bar for file operations, saving, and accessing data panel.
* **Profile and Help:** Access to account settings, preferences, and help resources.
* **Toolbar:** Contains tools organized by workspaces (Design, Generative Design, Render, Animation, Simulation, Manufacture, Drawing). The "Solid" and "Sketch" tabs are primary in the Design workspace.
* **ViewCube:** Located in the top-right corner, used for orienting the 3D model.
* **Browser:** On the left, organizes all design elements (bodies, sketches, components, joints, document settings, origin planes/axes).
* **Canvas (3D Environment):** The main workspace where you create and interact with your 3D models.
* **Timeline:** At the bottom, records the history of operations, allowing parametric modifications.
* **Navigation Bar & Display Settings:** At the bottom of the canvas, provides tools for pan, zoom, orbit, and visual style adjustments.
